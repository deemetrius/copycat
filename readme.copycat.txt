copycat: Quick copy stored text to clipboard from popup menu (tray icon).

* Left click on tray icon to show popup menu.
* Right click on tray icon to toggle main window.

List contains Name - Value pairs.
When you click on popup menu item - its Value will be copied to clipboard.

All settings of this program are stored inside this folder: "%AppData%\dt_copycat"

platform: Only windows 64 bits.
